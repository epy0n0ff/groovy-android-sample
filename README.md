Groovy Android Sample
================

概要
----

GroovyでAndroidのアプリを書くサンプルです。

使い方
----

groodroidが必要になるので以下のリポジトリからソースコードを取得してjarをビルドしてください。

https://github.com/groovy/groovy-core
